# PDF_tools

1. 一个PDF工具，可以拆分PDF和OCR识别PDF。
2. OCR识别仅限与英文，中文识别准确率几乎为零。
3. 运行代码序安装相应Python库和Tesseract。
4. 可安装exe软件直接使用。
